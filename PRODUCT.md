# BetterPrompt 功能文档

**输入优化** — 删掉 prompt 里没用的词和句子

**结构控制** — 控制 prompt 整体的长度和格式

| 功能名称 | 功能说明 | 类型 | 阶段 |
|---------|---------|------|------|
| 问候语清除 | 删除 prompt 开头的问候语，例如 "Hello" 或 "Hi There" | 输入优化 | 第一阶段 |
| 任务识别 | 自动判断这个 prompt 是在问什么类型的问题 | 输入优化 | 第一阶段 |
| 啰嗦文字替换 | 把常见的冗长表述转换成更简短的表达 | 输入优化 | 第一阶段 |
| 空白清理 | 删除多余的空行和空格 | 输入优化 | 第一阶段 |
| 长度限制 | 超过设定字数上限时自动截断 | 结构控制 | 第一阶段 |
| 格式压缩 | 把格式指令换成更短的符号表示 | 结构控制 | 第一阶段 |
| 废话删除 | 删除结尾没有实际意义的客套句子 | 结构控制 | 第一阶段 |
