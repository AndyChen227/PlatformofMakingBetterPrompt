# BetterPrompt 功能文档

**第一阶段（Level 1）** 处理输入本身的内容，删除和替换无用的词句；**第二阶段（Level 2）** 控制输出的结构和长度，约束 prompt 的格式和体积。

## 输入优化 — 删掉 prompt 里没用的词和句子

| 功能名称 | 英文名称 | 功能说明 | 类型 | 阶段 |
|---------|---------|---------|------|------|
| 问候语清除 | Input Cleaner | 删除 prompt 开头的问候语，例如 "Hello" 或 "Hi There" | 输入优化 | 第一阶段 |
| 任务识别 | Task Analyzer | 自动判断这个 prompt 是在问什么类型的问题 | 输入优化 | 第一阶段 |
| 啰嗦文字替换 | Semantic Compressor | 把常见的冗长表述转换成更简短的表达 | 输入优化 | 第一阶段 |
| 空白清理 | Structure Minimizer | 删除多余的空行和空格 | 输入优化 | 第一阶段 |
| 标点规范化 | Punctuation Normalizer | 删除重复标点符号，例如 "!!!" → "!"，规范省略号用法 | 输入优化 | 第一阶段 |
| 数字标准化 | Number Normalizer | 把英文数字和百分比表达转换成阿拉伯数字，例如 "two hundred" → "200" | 输入优化 | 第一阶段 |

## 结构控制 — 控制 prompt 整体的长度和格式

| 功能名称 | 英文名称 | 功能说明 | 类型 | 阶段 |
|---------|---------|---------|------|------|
| 长度限制 | Length Control | 超过设定字数上限时自动截断 | 结构控制 | 第一阶段 |
| 格式压缩 | Format Control | 把格式指令换成更短的符号表示 | 结构控制 | 第一阶段 |
| 废话删除 | Redundancy Suppressor | 删除结尾没有实际意义的客套句子 | 结构控制 | 第一阶段 |
